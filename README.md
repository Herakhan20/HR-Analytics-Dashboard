 HR Analytics Dashboard (Power BI)

 📌 Project Overview

This dashboard analyzes employee data using Power BI.

 📊 KPIs Included

* Total Employees
* Average Salary
* Attrition Count
* Attrition Rate
* Average Tenure

 📈 Visualizations

* Employees by Department
* Salary by Department
* Gender Distribution
* Attrition by Department
* Department Filter (Slicer)

 🛠 Tools Used

* Power BI
* DAX
* Data Modeling
